# 7.9  Exercises

## Exercise 1  
Write a number guessing game where the program picks a number between 1 and 100, inclusive (meaning 1 and 100 are valid numbers).  The human / user tries to guess the number, and the program gives feedback: correct, too low, or too high.

Use the template - *guess_the_number_picker.py*


## Exercise 2
Write a number guessing game similar to the first one.  This time, the human / user picks a number from 1 to 100, and the program has to guess the number.  The human gives the program feedback: (c) correct, (l) too low, (h) too high.

You can start with a "dumb" program to start with that just picks a random number each time, like a toddler, with no sense of how to pick a better 

Use the template - *guess_the_number_guesser.py*
